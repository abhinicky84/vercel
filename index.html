Division of Responsibility

What AI Delivers — and Where
Verticurl Dev Team Takes Over

The AI Agent is a powerful accelerator. Here's exactly how the work is divided between the assistant and the Verticurl Dev Team.

✅ What the AI Agent Does

  • Analyses the prompt or reference URL to understand the form's purpose
  • Identifies all required input fields and appropriate field types
  • Applies field-level validations automatically
  • Scaffolds the form using the selected default template and theme
  • Generates a Submit CTA with the defined submission link

🛠️ What Verticurl Dev Team Delivers

  • Advanced look & feel beyond default themes and base CSS
  • Custom branding, colours, typography, and layout refinements
  • Backend data submission logic and API integrations
  • Complex conditional logic and multi-step form flows
  • Accessibility enhancements and performance tuning
💡 Key Insight

The AEM Forms AI Assistant is a powerful accelerator — it handles the structural heavy lifting so the Verticurl Dev Team can focus purely on customisation, integration, and delivering exceptional user experiences.
