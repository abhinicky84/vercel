Architecture Strategy

Interactive Experience Integration Framework

Executive Directive: Transition the web estate from passive feature-listing templates to active, self-guided workflow models. The goal is to let prospects validate Octave platform capability (EAM, Aria Agentic Layer, Forte 3D) without sales onboarding friction.

The Integration Candidates

  • Figma Design Sandbox: High-fidelity interaction layer designed to embed core layout structures directly from design workspaces.
  • Consensus Automated Demos: Enterprise presales platform delivering personalized, multi-branching video interactive scenarios.
  • Arcade Interactive Walkthroughs: Lightweight marketing browser-capture solution driving annotated product flows.

Unified Component Positioning Strategy

  • Place components midway down target product pages to maximize visual value.
  • Position after core capability description, preceding analyst & G2 verification cards.
  • Utilize a "Facade Pattern" wrapper to eliminate initial script loading overhead.

Direct Comparative Trade-offs

1. Arcade (The GTM Standard)

Operational Advantages

  • Ultra-rapid deployment; build and launch annotated tours in under 15 minutes.
  • Outstanding browser loading performance using optimized static assets.
  • Simple, non-technical maintenance interface for quick content updates.

Operational Risks

  • Limited support for highly customized user input simulation.
  • Lacks native, deep account-level buying committee identification.

2. Consensus Presales Platform

Operational Advantages

  • Surfaces the buyer committee by tracking viewer share paths.
  • Adapts demo scenarios based on self-selected target personas.
  • Native API syncs direct account interaction signals to Salesforce.

Operational Risks

  • Requires heavy up-front asset configuration and structural planning.
  • Cost-prohibitive for simple awareness marketing templates.

3. Figma Widget Engine

Operational Advantages

  • Synchronous updates directly mirror the design workspace instantly.
  • Maintains absolute design systems compliance across teams.
  • Zero ongoing platform licensing costs.

Operational Risks

  • Severe loading times; blocks key rendering threads (hurts SEO/LCP).
  • No guided tooltips or helper instructions for non-technical buyers.
  • Complete absence of analytics feedback or user tracking metrics.
Dev Engineering

VueJS & Contentful Implementation Specs

For Stephen and Michelle, we establish a standardized component schema to keep design templates safe, maintain performance, and avoid code redeployments.

Contentful Content Schema Definition

  • Content Type: InteractiveDemoBlock (Custom Component).
  • CMS Form Inputs:
    • provider: Dropdown field (arcade | consensus).
    • embedUrl: Text validation field requiring secure HTTPS domain source.
    • facadeImage: Media asset pointer for high-resolution web placeholder.
    • aspectRatio: Dropdown locking element layout (16:9 | 4:3).

Vue.js Lazy-Rendering Optimization (Facade Pattern)

To secure our Core Web Vitals, the browser must not fetch the dynamic iframe during initial page parsing. We load a lightweight image facade that hot-loads the target interactive frame on demand:

const formattedUrl = computed(() => { const url = new URL(props.embedUrl); const params = new URLSearchParams(window.location.search); const trackingTags = ['utm_source', 'utm_medium', 'utm_campaign']; trackingTags.forEach(tag => { if (params.has(tag)) url.searchParams.set(tag, params.get(tag)); }); return url.toString(); });
Business Enablement

Marketer Empowerment & Speed-to-Market

To meet executive timelines without burdening the engineering backlog, we recommend deploying a hybrid Arcade and Consensus GTM strategy.

The dual marketing strategy designed for rapid activation:

  • Phase 1: Rapid GTM Execution (Arcade Embeds): Empowers our product marketing team to capture, edit, and publish target walkthrough workflows within 1 day. Stephen’s Contentful schema lets marketers swap demo paths in the CMS without dev deployments.
  • Phase 2: High-Value Pipeline Conversion (Consensus integration): Focuses on late-stage product pages and account landing sites. Delivers personalized branching paths, passing enriched buyer profiles directly to our CRM.

Key Strategic Benefits

  • 100% Development Decoupling: Marketing updates, updates paths, and swaps copy inside the CMS without engineering code changes.
  • Maximum Performance Security: Lazy-loaded code architecture maintains PageSpeed scores and protects organic rank visibility.
  • Attribution Preservation: Parent-to-iframe parameter passing keeps tracking, UTM sources, and analytics logs intact.
