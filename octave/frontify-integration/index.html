Architecture Strategy

Native Frontify Previews Inside Contentful

Executive Summary: Streamline the editorial workflow by integrating Frontify’s secure brand-asset CDN directly into the Contentful editing workspace. This eliminates manual download and re-upload workflows, establishing a verified visual asset pipeline with zero layout shifts.

The Operational Problem Statement

  • Broken Editorial Loops: Content editors currently link to assets blindly, copying URLs from Frontify without on-screen validation inside Contentful.
  • Asset Disconnects: Changes made in the master brand hub do not propagate safely, leading to broken assets and manual versioning errors.
  • Layout Stability Risk: Lack of structured dimension metadata causes Cumulative Layout Shifts (CLS) upon asset load, degrading SEO performance.

The Strategic Integration Solution

  • Natively Rendered Previews: Integrates Frontify Finder v2 into the Contentful App Framework to display true thumbnail images directly inside the content model.
  • Automatic Dimensions Mapping: Synchronously parses layout dimensions (width and height ratios) from the dynamic asset payload.
  • Version-Controlled CDN Links: Connects to unique, revision-specific CDN URLs to guarantee layout and performance safety during platform updates.

Technical Specification

JSON Metadata Binding & Rendering Architecture

To avoid custom middleware development, we implement an established market solution that binds asset metadata directly to the Contentful field schema.

Rich Metadata Schema (Payload Standard)

Rather than managing raw asset URL strings, the Contentful custom field stores a structured object carrying dimensions and asset credentials:

{ "assetId": "frontify_asset_88bc27", "title": "Aria Platform Dashboard Preview", "cdnUrl": "https://octave-cdn.frontify.com/api/...", "width": 1920, "height": 1080, "focalPoint": "[0.50, 0.45]" }

Performance-First Vue 3 Component Blueprint

Stephen and Michelle can implement this optimized rendering component to dynamically manage aspect ratios and enforce zero Cumulative Layout Shift:

<template> <div :style="{ aspectRatio: `${payload.width} / ${payload.height}` }" class="media-box"> <img :src="payload.cdnUrl" loading="lazy" alt="payload.title" /> </div> </template>
